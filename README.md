# Faby Vanyo — Portfolio Website

Personal portfolio and work showcase for Faby Vanyo, Senior Front-End Engineer & AI Product Builder.

## Built With
- Vanilla HTML5, CSS3, JavaScript (no frameworks, no build step needed)
- Google Fonts: Inter + Playfair Display
- Fully responsive · Dark theme · Scroll animations

## Sections
- **Hero** — Title, tagline, and CTA
- **About** — Summary + stats
- **Independent Work & AI Products** — FabVista Studio, Fab Digital Studio, AI Workflow Systems, Payment Integrations
- **Professional Experience** — Full timeline: EVERSANA, MoodMD, COTY/Wella, PlanetArt, Gifts.com, and earlier roles
- **Skills & Tools** — All tech grouped by category
- **Education & Certifications** — CSUN, SMC, Merit College, Loyola Law School
- **Contact** — Email, phone, LinkedIn

## Deploy
Drop the files on any static host (Netlify, Vercel, GitHub Pages) — no build step required.

For GitHub Pages: enable Pages from the `main` branch root in repo Settings.
